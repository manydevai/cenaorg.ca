# CENA.ORG 🍁

Comunidade de Educação e Networking Angolana au Canada. 

A React-based web application to empower the Angolan and Portuguese-speaking diaspora in Canada through strategic education, entrepreneurship, and global networking.

### Tech Stack
- Frontend: React (Vite)
- Language: TypeScript
- Styling: TailwindCSS 
- Animations: Framer Motion
- UI Components: Radix UI

### Quick Start 🚀

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Build production bundle:
```bash
npm run build
```

The application will be accessible locally at `http://localhost:5173`.

### Supported Languages
The codebase manages internationalization across 3 languages: 
- 🇫🇷 French
- 🇬🇧 English
- 🇵🇹 Portuguese
